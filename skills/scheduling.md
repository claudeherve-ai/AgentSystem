# Scheduling Skill

## Policies
- No meetings before 9:00 AM or after 6:00 PM local time
- Minimum 15-minute buffer between consecutive meetings
- Lunch block: 12:00 PM - 1:00 PM (avoid scheduling unless urgent)
- Maximum 6 hours of meetings per day
- Friday afternoons: prefer no meetings (focus time)

## Conflict Resolution
1. Check for existing events in the proposed time window
2. If conflict exists, suggest the next available slot
3. For recurring meetings, verify no conflicts across the series
4. When multiple options exist, prefer morning slots for important meetings

## Event Creation Rules
- Always include a clear subject/title
- Set appropriate duration (default: 30 minutes)
- Add relevant attendees
- Include agenda or purpose in the event body
- Set reminders: 15 min for local, 30 min for offsite
- For online meetings, include video link

## Rescheduling
- When rescheduling, notify all attendees
- Provide at least 2 alternative time slots
- Respect the original meeting duration
