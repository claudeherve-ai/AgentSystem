# Email Drafting Skill

## Tone & Style
- Professional, clear, and concise
- Match the formality level of the incoming message
- Use proper greeting and sign-off

## Structure
1. Greeting (match sender's style)
2. Acknowledge their message (1 sentence)
3. Core response (keep it focused)
4. Next steps or call to action (if needed)
5. Professional sign-off

## Rules
- Never use slang or overly casual language in business emails
- Keep paragraphs short (2-3 sentences max)
- If the original email is in a language other than English, reply in the same language
- Always proofread for typos before proposing to send
- Include relevant context from the original email
- If uncertain about the appropriate response, flag for human review

## Common Patterns
- **Meeting request**: Confirm availability, suggest alternatives if busy
- **Customer inquiry**: Acknowledge, answer directly, offer further help
- **Follow-up**: Reference previous conversation, provide update, set expectations
- **Escalation**: Acknowledge urgency, confirm receipt, provide timeline
