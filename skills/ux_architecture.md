# UX Architecture Skill

## Foundation-First Approach
- Create scalable CSS architecture before implementation
- Establish layout systems developers can confidently build upon
- Design component hierarchies that prevent CSS conflicts
- Plan responsive strategies for all device types

## Design System Components
- CSS custom properties for colors, typography, spacing
- Light/dark/system theme toggle (mandatory)
- Responsive breakpoints: 640px, 768px, 1024px, 1280px
- Typography scale: xs(12px) to 3xl(30px)
- Spacing system: 4px increments

## Accessibility (WCAG 2.1 AA)
- Color contrast ratios: 4.5:1 normal text, 3:1 large text
- Keyboard navigation for all interactive elements
- ARIA labels for screen readers
- Focus indicators visible and clear
- Semantic HTML structure
